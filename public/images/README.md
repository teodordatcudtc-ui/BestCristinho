# Folder pentru Logo

## Cum să adaugi logo-ul

1. **Adaugă imaginea logo-ului** în acest folder cu numele: `logo.png`

### Formate acceptate:
- `logo.png` (recomandat)
- `logo.jpg` sau `logo.jpeg`
- `logo.svg` (pentru logo-uri vectoriale)

### Dimensiuni recomandate:
- **Lățime:** 200-300px
- **Înălțime:** 50-100px
- **Format:** PNG cu fundal transparent (recomandat) sau JPG

### Notă:
- Logo-ul va fi afișat automat în header după ce adaugi fișierul
- Dacă logo-ul nu există, se va afișa textul "Best Cristinho" ca fallback
- Logo-ul este optimizat automat de Next.js pentru performanță

### Exemplu:
```
public/images/logo.png
```
